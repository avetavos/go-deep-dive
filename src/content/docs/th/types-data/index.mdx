## Types และ Data Structures ใน Go

โมดูลนี้ครอบคลุม data structures หลักของ Go ได้แก่ arrays ที่มีขนาดคงที่, slices ที่ขยายตัวได้, maps สำหรับ key-value storage, และ structs สำหรับการจัดกลุ่ม fields ที่เกี่ยวข้อง นอกจากนี้ยังสอน pointer semantics และความแตกต่างระหว่าง value types กับ reference types ซึ่งเป็นแนวคิดสำคัญใน Go

> โมดูลนี้กำลังจะมาเร็ว ๆ นี้ คอยติดตามบทเรียนเกี่ยวกับ arrays, slices, maps, structs, และ pointers
