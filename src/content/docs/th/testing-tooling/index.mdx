## Testing และ Tooling ใน Go

Go มี testing framework ในตัวที่ไม่ต้องพึ่ง third-party libraries โมดูลนี้สอนรูปแบบ table-driven tests ซึ่งเป็น idiomatic ของ Go, การเขียน benchmarks ด้วย `testing.B`, การใช้ race detector (`go test -race`) เพื่อตรวจจับ data races, และเครื่องมือ toolchain สำคัญ เช่น `go vet`, `gofmt`, และ `staticcheck` สำหรับรักษาคุณภาพโค้ด

> โมดูลนี้กำลังจะมาเร็ว ๆ นี้ คอยติดตามบทเรียนเกี่ยวกับ go test, table-driven tests, benchmarks, race detector, และ go vet
