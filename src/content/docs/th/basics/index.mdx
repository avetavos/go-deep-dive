## พื้นฐานและ Syntax ของ Go

โมดูลนี้สอนรากฐานของภาษา Go ตั้งแต่วิธีประกาศ variables และ constants ไปจนถึง type system แบบ static, โครงสร้างควบคุม (`if`, `for`, `switch`), การเขียนและเรียก functions พร้อม multiple return values, และการจัดการ packages และ imports ตามแบบฉบับ Go

> โมดูลนี้กำลังจะมาเร็ว ๆ นี้ คอยติดตามบทเรียนเกี่ยวกับ variables, types, control flow, functions, และ packages
