## ปรัชญาการออกแบบของ Go

Go ถูกออกแบบโดยทีมที่ Google เพื่อแก้ปัญหาจริงในการพัฒนา software ขนาดใหญ่ ปรัชญาหลักมีสี่ข้อ:

**ความเรียบง่าย (Simplicity)** — Go เลือกความเรียบง่ายเหนือความหลากหลายของ feature ทุก keyword และ syntax มีเหตุผลที่ชัดเจน ภาษามี keyword เพียง 25 คำ ทำให้เรียนรู้ได้เร็วและอ่าน code ของคนอื่นได้ง่าย

**เวลา compile เร็ว (Fast Compile)** — Go compile เป็นวินาที ไม่ใช่นาที แม้สำหรับ codebase ขนาดใหญ่ ทำให้ feedback loop สั้นและ iteration เร็ว

**วิธีเดียวที่ชัดเจน (One Obvious Way)** — สำหรับปัญหาส่วนใหญ่ มักมีวิธีหนึ่งที่ Go แนะนำ สิ่งนี้ทำให้ codebase ขนาดใหญ่อ่านสม่ำเสมอ เพราะทุกคนเขียนด้วยรูปแบบเดียวกัน

**Explicit มากกว่า Implicit** — Go บังคับให้คุณระบุประเภทและการแปลงอย่างชัดเจน ไม่มี magic type coercion ที่ซ่อนอยู่

## โมดูลนี้ครอบคลุมอะไร

โมดูล Basics จะพาคุณผ่านรากฐานของภาษา:

- **Packages และ Imports** — วิธีจัดการ code ใน package และ import dependency
- **Variables และ Types** — การประกาศตัวแปร, zero values, types พื้นฐาน, constants, และ `iota`
- **Control Flow** — `if`, `for` (loop keyword เดียวของ Go), `switch`, และ `range`
- **Functions** — multiple return values, `defer`, closures, และ first-class functions

## โปรแกรมแรก

ด้านล่างคือโปรแกรม Go ที่แสดงให้เห็น syntax พื้นฐาน: การประกาศตัวแปรด้วย `:=`, types หลายชนิด, และการ print ผลลัพธ์

```go
package main

import "fmt"

func main() {
    name := "Go"
    version := 1.22
    fast := true
    fmt.Println("Hello from", name)
    fmt.Println("Version:", version)
    fmt.Println("Fast compile:", fast)
}
```

รันด้านล่างเพื่อดู output ทุก concept ที่ใช้ที่นี่จะถูกอธิบายอย่างละเอียดในบทเรียนต่อไป:

export const indexCode = `package main

import "fmt"

func main() {
\tname := "Go"
\tversion := 1.22
\tfast := true
\tfmt.Println("Hello from", name)
\tfmt.Println("Version:", version)
\tfmt.Println("Fast compile:", fast)

\t// Integer arithmetic is exact; no implicit float promotion
\ta := 10
\tb := 3
\tfmt.Println("10 / 3 =", a/b)
\tfmt.Println("10 % 3 =", a%b)
}`;
