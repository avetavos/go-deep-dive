## Syntax พื้นฐานของ Function

Function ใน Go ประกาศด้วย keyword `func` ตามด้วยชื่อ, parameters, และ return types:

```go
func add(x int, y int) int {
    return x + y
}

// Parameters ที่มี type เดียวกันสามารถเขียนรวมได้
func add(x, y int) int {
    return x + y
}
```

## Multiple Return Values

Go functions สามารถ return หลายค่าได้ ซึ่งเป็น pattern มาตรฐานสำหรับ error handling:

```go
func divide(a, b float64) (float64, error) {
    if b == 0 {
        return 0, errors.New("division by zero")
    }
    return a / b, nil
}

result, err := divide(10, 3)
if err != nil {
    log.Fatal(err)
}
```

Pattern `(value, error)` ถือเป็น idiomatic Go — แทบทุก function ที่อาจล้มเหลวจะ return error เป็น value สุดท้าย

## Named Return Values

Return values สามารถตั้งชื่อได้ ทำให้สามารถใช้ **bare return** ได้:

```go
func minMax(nums []int) (min, max int) {
    min, max = nums[0], nums[0]
    for _, n := range nums[1:] {
        if n < min {
            min = n
        }
        if n > max {
            max = n
        }
    }
    return   // return min, max โดยไม่ต้องระบุ
}
```

ใช้ named returns เพื่อความชัดเจนใน function สั้น ๆ หลีกเลี่ยงใน function ยาวเพราะทำให้อ่านสับสน

## Variadic Functions

ใช้ `...T` เพื่อรับ arguments จำนวนไม่จำกัดของ type เดียวกัน:

```go
func sum(nums ...int) int {
    total := 0
    for _, n := range nums {
        total += n
    }
    return total
}

sum(1, 2, 3)       // ส่งแต่ละตัว
sum(1, 2, 3, 4, 5) // จำนวนเท่าไรก็ได้

existing := []int{1, 2, 3}
sum(existing...)   // spread slice ด้วย ...
```

## defer

`defer` เลื่อนการเรียก function ออกไปจนกว่า function ที่ครอบอยู่จะ return เหมาะสำหรับ cleanup:

```go
func readFile(path string) error {
    f, err := os.Open(path)
    if err != nil {
        return err
    }
    defer f.Close()   // จะถูกเรียกเมื่อ readFile return ไม่ว่าจะ return ปกติหรือ error
    // ... ใช้ f
}
```

**Arguments ของ deferred call ถูก evaluate ทันที** ณ จุดที่ `defer` statement ถูก execute ไม่ใช่ตอน call จริง:

```go
i := 0
defer fmt.Println(i)   // บันทึกค่า 0 ณ ตรงนี้
i = 100
// เมื่อ function return จะ print "0" ไม่ใช่ "100"
```

หลาย `defer` ในฟังก์ชันเดียวจะ execute แบบ LIFO (last in, first out)

## Closures และ First-Class Functions

Functions ใน Go เป็น first-class values — สามารถ assign ให้ตัวแปร, ส่งเป็น argument, หรือ return จาก function ได้:

```go
// Function เป็น value ใน variable
double := func(x int) int { return x * 2 }
fmt.Println(double(5)) // 10

// Function เป็น parameter
apply := func(f func(int) int, x int) int { return f(x) }
fmt.Println(apply(double, 3)) // 6
```

**Closure** คือ function ที่ "จำ" ตัวแปรจาก enclosing scope:

```go
func makeCounter() func() int {
    count := 0
    return func() int {
        count++      // capture ตัวแปร count จาก makeCounter
        return count
    }
}

counter := makeCounter()
fmt.Println(counter()) // 1
fmt.Println(counter()) // 2 — state ถูกเก็บไว้ใน closure
```
