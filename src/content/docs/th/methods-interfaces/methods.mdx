## Method คืออะไร?

**Method** คือฟังก์ชันที่มี *receiver* — parameter พิเศษที่ปรากฏระหว่าง keyword `func` กับชื่อฟังก์ชัน receiver ผูกฟังก์ชันนั้นเข้ากับ named type

```go
func (r Rectangle) Area() float64 {
    return r.Width * r.Height
}
```

ที่นี่ `r Rectangle` คือ receiver `Area` กลายเป็น method ของ `Rectangle` เรียกได้ว่า `rect.Area()`

## Value receiver vs Pointer receiver

Receiver สามารถเป็น value หรือ pointer ก็ได้:

```go
func (c Counter) Value() int  { return c.count }   // value receiver
func (c *Counter) Inc()       { c.count++ }        // pointer receiver
```

| | Value receiver `(t T)` | Pointer receiver `(t *T)` |
|--|------------------------|---------------------------|
| **เห็น** | สำเนาของค่า | ค่าต้นฉบับ |
| **แก้ไขได้?** | ไม่ได้ — แก้เฉพาะสำเนา | ได้ |
| **เรียกได้จาก** | ทั้ง `T` และ `*T` | ทั้ง `T` (addressable) และ `*T` |
| **ใช้เมื่อ** | อ่านอย่างเดียว; ขนาดเล็ก, copy ถูก | ต้องแก้ไข; struct ใหญ่; method set สม่ำเสมอ |

**ความสม่ำเสมอ:** ถ้า method ใด ๆ บน type ต้องการ pointer receiver ให้ใช้ pointer receiver กับ *ทุก* method ของ type นั้น การผสมกันทำให้เกิดปัญหา method set ที่ละเอียดอ่อนเมื่อ type ถูกใช้เป็น interface value

## Methods บน named type ใดก็ได้

Methods ไม่จำกัดแค่ structs คุณนิยาม methods บน named type ใด ๆ ใน package ของคุณได้:

```go
type Celsius float64

func (t Celsius) String() string {
    return fmt.Sprintf("%.1f°C", float64(t))
}
```

คุณไม่สามารถนิยาม method บน type จาก package อื่น (เช่น `func (s string) Upper()` ผิดกฎ) แต่การห่อมันใน named type ในท้องถิ่นใช้ได้เสมอ

## Method sets

**Method set** ของ type กำหนดว่า interfaces ใดที่ตอบสนองได้:

- Method set ของ `T` มี methods ทั้งหมดที่มี value receiver `(t T)`
- Method set ของ `*T` มี methods ทั้งหมด — ทั้ง `(t T)` และ `(t *T)`

นี่คือเหตุผลที่ interface ที่ต้องการ pointer-receiver method สามารถตอบสนองได้เฉพาะโดย `*T` ไม่ใช่ `T`

export const methodsCode = `package main

import "fmt"

type Counter struct {
\tcount int
}

// Pointer receiver — mutates the struct
func (c *Counter) Inc() {
\tc.count++
}

// Value receiver — read-only
func (c Counter) Value() int {
\treturn c.count
}

type Celsius float64

func (t Celsius) String() string {
\treturn fmt.Sprintf("%.1f\\u00b0C", float64(t))
}

func main() {
\tc := Counter{}
\tc.Inc()
\tc.Inc()
\tc.Inc()
\tfmt.Println(c.Value())

\ttemp := Celsius(36.6)
\tfmt.Println(temp.String())
}`;
