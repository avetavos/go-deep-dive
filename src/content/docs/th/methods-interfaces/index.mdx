## Methods และ Interfaces ใน Go

โมดูลนี้สอนวิธีแนบ behaviour ให้กับ types ด้วย methods (value receiver vs pointer receiver), การออกแบบ interfaces แบบ implicit satisfaction ที่เป็นเอกลักษณ์ของ Go, การประกอบ interfaces ขนาดเล็กเป็น interfaces ขนาดใหญ่, และการใช้ struct embedding เพื่อแชร์ implementation โดยไม่ต้องใช้ inheritance

> โมดูลนี้กำลังจะมาเร็ว ๆ นี้ คอยติดตามบทเรียนเกี่ยวกับ methods, interfaces, composition, และ embedding
