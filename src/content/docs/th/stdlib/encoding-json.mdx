## JSON ใน Go

Package `encoding/json` แปลงค่า Go เป็น JSON text (marshaling) และแปลง JSON text กลับเป็นค่า Go (unmarshaling) ฟังก์ชันหลักสองตัวคือ:

```go
data, err := json.Marshal(v)       // Go value → []byte (JSON)
err        = json.Unmarshal(data, &v) // []byte (JSON) → Go value
```

`Marshal` รับค่าใดก็ได้และคืน `[]byte` `Unmarshal` เขียนลงในค่าที่ชี้โดย argument ที่สอง — ส่ง pointer เสมอเพื่อให้ฟังก์ชันแก้ไขตัวแปรได้

## Struct tags

**Struct tag** คือ raw-string literal บน field ของ struct ที่ให้ metadata Package `encoding/json` อ่าน tag `json:"name"` เพื่อควบคุมการ serialize:

```go
type User struct {
    Name  string `json:"name"`
    Email string `json:"email,omitempty"`
    Age   int    `json:"age"`
}
```

syntax ของ tag คือ `` `json:"<name>[,options]"` `` option ที่พบบ่อย:

| Tag | ผล |
|-----|--------|
| `json:"name"` | ใช้ `name` เป็น JSON key แทนชื่อ field |
| `json:"name,omitempty"` | ละเว้น field นี้ใน output เมื่อค่าเป็น zero value |
| `json:"-"` | ละเว้น field นี้เสมอ |

## Nested structs

Field ของ struct สามารถเป็น struct ได้เช่นกัน `json.Marshal` จะ recurse เข้าไปโดยอัตโนมัติ:

```go
type Address struct {
    City    string `json:"city"`
    Country string `json:"country"`
}

type User struct {
    Name    string  `json:"name"`
    Age     int     `json:"age"`
    Address Address `json:"address"`
}
```

JSON ที่ได้จะฝัง address object ไว้ภายใต้ key `"address"`

## map[string]any — การ unmarshal แบบยืดหยุ่น

เมื่อยังไม่มี struct type ล่วงหน้า ให้ unmarshal ลงใน `map[string]any` แต่ละค่า JSON จะกลายเป็น Go counterpart ที่เป็นธรรมชาติ: `string`, `float64`, `bool`, `[]any`, `map[string]any` หรือ `nil`

```go
var m map[string]any
json.Unmarshal([]byte(`{"lang":"Go","version":1.22}`), &m)
fmt.Println(m["lang"])    // Go
fmt.Println(m["version"]) // 1.22  (float64)
```

export const encodingJsonCode = `package main

import (
\t"encoding/json"
\t"fmt"
)

type Address struct {
\tCity    string \`json:"city"\`
\tCountry string \`json:"country"\`
}

type User struct {
\tName    string  \`json:"name"\`
\tAge     int     \`json:"age"\`
\tEmail   string  \`json:"email,omitempty"\`
\tAddress Address \`json:"address"\`
}

func main() {
\t// Marshal a struct — Email is set, so it appears
\tu := User{
\t\tName:  "Alice",
\t\tAge:   30,
\t\tEmail: "alice@example.com",
\t\tAddress: Address{
\t\t\tCity:    "Bangkok",
\t\t\tCountry: "Thailand",
\t\t},
\t}
\tdata, err := json.Marshal(u)
\tif err != nil {
\t\tfmt.Println("marshal error:", err)
\t\treturn
\t}
\tfmt.Println(string(data))

\t// omitempty: Email is empty, so it is omitted
\tu2 := User{Name: "Bob", Age: 25, Address: Address{City: "London", Country: "UK"}}
\tdata2, _ := json.Marshal(u2)
\tfmt.Println(string(data2))

\t// Unmarshal back into a struct
\tvar result User
\terr = json.Unmarshal(data, &result)
\tif err != nil {
\t\tfmt.Println("unmarshal error:", err)
\t\treturn
\t}
\tfmt.Printf("Name: %s, City: %s\\n", result.Name, result.Address.City)

\t// map[string]any — flexible unmarshaling
\traw := \`{"lang":"Go","version":1.22,"stable":true}\`
\tvar m map[string]any
\t_ = json.Unmarshal([]byte(raw), &m)
\tfmt.Println(m["lang"], m["version"], m["stable"])
}`;
