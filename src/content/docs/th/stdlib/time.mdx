## time.Time และ time.Duration

Package `time` มี type หลักสองตัว:

- **`time.Time`** — ช่วงเวลาหนึ่งที่มีความแม่นยำระดับ nanosecond พร้อม location (timezone) เสมอ
- **`time.Duration`** — signed 64-bit integer แทนความยาวของเวลาในหน่วย nanosecond ค่าคงที่อย่าง `time.Hour`, `time.Minute`, `time.Second` และ `time.Millisecond` ทำให้การคำนวณอ่านง่าย

```go
d := 2*time.Hour + 30*time.Minute + 15*time.Second
fmt.Println(d) // 2h30m15s
```

## การสร้าง time.Time

ในโค้ด production จะเรียก `time.Now()` เพื่อรับเวลาปัจจุบัน สำหรับการทดสอบและตัวอย่างที่ต้องการผลลัพธ์คงที่ให้ใช้ `time.Date`:

```go
t := time.Date(2024, 1, 2, 15, 4, 5, 0, time.UTC)
// year, month, day, hour, min, sec, nanosec, location
```

ค่าของ `time.Month` (ตั้งแต่ `time.January` ถึง `time.December`) เป็น typed constant แต่ก็สามารถส่ง integer literal (1–12) ได้เพราะสามารถ assign ให้ `time.Month` ได้

## การฟอร์แมต — reference layout

Go ใช้ช่วงเวลาอ้างอิงที่เป็นรูปธรรมแทนรหัสฟอร์แมตแบบนามธรรมอย่าง `YYYY-MM-DD` ช่วงเวลาอ้างอิงคือ:

```
Mon Jan 2 15:04:05 MST 2006
```

หากต้องการฟอร์แมตเวลา ให้เขียนช่วงเวลาอ้างอิงในรูปแบบที่ต้องการ:

```go
t.Format("2006-01-02")             // "2024-01-02"
t.Format("15:04:05")               // "15:04:05"
t.Format("02 Jan 2006 15:04 MST")  // "02 Jan 2024 15:04 UTC"
t.Format(time.RFC3339)             // "2024-01-02T15:04:05Z"
```

## การ Parse

`time.Parse` เป็นการทำงานตรงข้ามกับ `Format` — รับ layout และ string:

```go
parsed, err := time.Parse("2006-01-02", "2024-06-15")
// parsed คือเที่ยงคืน UTC ของวันที่ 15 มิถุนายน 2024
```

string ของ layout ต้องตรงกับ component ของช่วงเวลาอ้างอิงที่ใช้ — ตัวอักษรเดียวกันในตำแหน่งเดียวกัน

## การคำนวณ — Add และ Sub

```go
deadline := t.Add(72 * time.Hour)        // 3 วันต่อมา
diff     := deadline.Sub(t)              // time.Duration: 72h0m0s
before   := deadline.Before(t)          // false
after    := deadline.After(t)           // true
```

`Add` รับ `Duration` และคืน `Time` `Sub` รับ `Time` สองค่าและคืน `Duration` ระหว่างทั้งสอง

export const timeCode = `package main

import (
\t"fmt"
\t"time"
)

func main() {
\t// Fixed time — deterministic in any interpreter
\tt := time.Date(2024, 1, 2, 15, 4, 5, 0, time.UTC)
\tfmt.Println(t)

\t// Formatting with the reference layout
\tfmt.Println(t.Format("2006-01-02"))
\tfmt.Println(t.Format("15:04:05"))
\tfmt.Println(t.Format("Mon, 02 Jan 2006 15:04:05 MST"))
\tfmt.Println(t.Format(time.RFC3339))

\t// Parsing
\tparsed, err := time.Parse("2006-01-02", "2024-06-15")
\tif err == nil {
\t\tfmt.Println(parsed.Format("January 2, 2006"))
\t}

\t// Arithmetic
\tdeadline := t.Add(72 * time.Hour)
\tfmt.Println(deadline.Format("2006-01-02 15:04:05"))

\tdiff := deadline.Sub(t)
\tfmt.Println(diff)

\t// Duration construction
\td := 2*time.Hour + 30*time.Minute
\tfmt.Println(d)

\t// Accessing date parts
\tfmt.Println(t.Year(), t.Month(), t.Day())
\tfmt.Println(t.Hour(), t.Minute(), t.Second())
}`;
