## Standard Library ของ Go

Go มาพร้อม standard library ที่ครอบคลุมและออกแบบมาอย่างดี โมดูลนี้สำรวจ packages ที่ใช้บ่อยที่สุด: `fmt` สำหรับ formatted I/O, `strings` และ `strconv` สำหรับจัดการข้อความ, `io` และ `bufio` สำหรับ stream processing, `net/http` สำหรับสร้าง HTTP clients และ servers, `encoding/json` สำหรับ serialization, และ `time` สำหรับจัดการวันเวลา

> โมดูลนี้กำลังจะมาเร็ว ๆ นี้ คอยติดตามบทเรียนเกี่ยวกับ fmt, strings, io, net/http, encoding/json, และ time
