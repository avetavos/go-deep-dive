## Testing & Tooling

Go's testing story is built into the language itself — no test framework needed. This module covers writing table-driven tests with subtests (`t.Run`), benchmarks with `b.Loop()` (Go 1.24+), using the race detector (`go test -race`) to catch data races at CI time, and the static analysis tools `go vet` and `golangci-lint` that enforce correctness and style before code review.
