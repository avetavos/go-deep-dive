## Basics & Syntax

This module covers the complete syntactic foundation of Go. You will learn how to declare variables with `var` and `:=`, work with Go's built-in types (`int`, `string`, `bool`, `float64`), write functions with multiple return values, use `if`, `for`, and `switch`, and organise code into packages. By the end you will be able to read and write any idiomatic Go program from scratch.
