## Types & Data

This module digs into Go's composite types. You will learn the difference between arrays and slices, how slice headers work under the hood (`len`, `cap`, backing array), how maps are hash tables with a simple API, how structs group related fields, and how pointers give you control over value vs reference semantics. Understanding these types deeply is the key to writing efficient, correct Go.
