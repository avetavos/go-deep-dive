## Errors & Generics

This module covers Go's explicit error model — errors are values, not exceptions — and the modern generics system added in Go 1.18. You will learn to wrap errors with context using `fmt.Errorf("%w", err)`, define custom error types that carry structured data, unwrap error chains with `errors.Is` and `errors.As`, and write generic functions and data structures using type parameters and constraints (`~int`, union types, `comparable`).
