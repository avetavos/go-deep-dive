## Standard Library

Go ships with a rich standard library that covers most of what you need without third-party dependencies. This module tours the packages you will reach for daily: `fmt` for formatted I/O, `strings` and `strconv` for text processing, `io` and `bufio` for streaming data, `net/http` for building and calling HTTP services, `encoding/json` for marshalling and unmarshalling, and `time` for durations, tickers, and timers.
