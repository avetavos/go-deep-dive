## Methods & Interfaces

This module covers one of Go's most powerful design ideas: interfaces are satisfied implicitly. You will learn to define methods on any named type, design small focused interfaces (the Go proverb: "the bigger the interface, the weaker the abstraction"), use struct embedding for composition instead of inheritance, and work with the standard `io.Reader` and `io.Writer` interfaces that underpin almost all I/O in Go.
